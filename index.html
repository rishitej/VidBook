About

This web application aims at providing a online video library, that gives users free access to videos sorted according to their categories. To view the videos, you can navigate to Categories section as shown in image 2 in the slidebar.

Each category has an "Add Video here!" button which enables to add your favourite videos(according to their respective category) . Also, we provide few intresting videos for the same. Have Fun!

You can also subscribe to us for latest updates and brand new features before others do. You can input your details in the Subscribe section below as shown in image 3 in slidebar.

This web application is built using the standard resources of html, css and javascript in Frontend; PHP and mySQL for creating form and storing data in backend. For more information check Tech Used section.
